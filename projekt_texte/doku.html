Projekt Beschreibung:

Es wird in einer Partnerarbeit eine 3- Tier WEB Applikation definiert und selbständig entwickelt.

Einzusetzende Technologien:

  • MySQL ist eine Sprache für die Datenbanken, wir Habens gewählt weil schon wussten, wie man das benutzt.
  • PHP ist eine Programmiersprache und wird für die Funktionalität der Website benötigt, z.B. Erstellen eines Login oder eines Accounts. PHP wurde gewählt, weil es als Schulstandard empfohlen wurde.
  • HTML für die Website, es ist eine Skript Sprache um Website zu kreieren. Wir Habens gewählt weil es nichts anderes gibt um Webseiten zu kreieren.
  • CSS ist ein Teil von HTML, mit dem kann man Webseiten stylen. Wir Habens gewählt weil es zu HTML passt
  • JS wir für Skripts benutzt, die direkt in die Webseite sind. Gewählt weil es gut für Scrips ist.
